# 初測程式簡介

此程式透過分析 CSV 檔案中的弱點資料，生成包含以下資訊的 Excel 報表：

- **風險數量列表**：統計每台主機的不同風險等級數量。
- **弱點列表**：篩選 `Risk >= Low` 的弱點並依風險等級排序。
- **弱點說明表格**：將 `Risk >= Low` 的弱點資訊轉置為橫向格式。
- **弱點數量**：統計不同風險等級的弱點數量。

## 所需資料

輸入檔案為一個包含以下欄位的 CSV 檔案：

- `Host`
- `Plugin ID`
- `Risk`
- `Name`
- `Description`
- `Solution`
- `See Also`
- `CVE`
